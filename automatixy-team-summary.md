# צוות הפיתוח של רומן — Automatixy AI Engineering Unit

## מה הצוות יודע לעשות

### פיתוח קוד מלא
- מקבל טיקט → בונה אותו על worktree מבודד
- Frontend, Backend, Database, DevOps — הכל בפנים
- לטיקט גדול — מפצל לכמה מפתחים במקביל

### בקרת איכות אוטומטית
- Code Review — בודק כל שינוי לפני merge
- Test Engineer — כותב טסטים לכל שינוי קוד
- QA Engineer — מריץ את האפליקציה ומחפש באגים בזמן אמת (UX, runtime, accessibility)

### אבטחה
- Security Engineer — חוסם merge אם מוצא חורי אבטחה (secrets, injection, dependencies מסוכנות)

### פריסה לפרודקשיון
- Release Manager — מאשר שהגרסה מוכנה (build, migrations, env, deploy)
- SRE — מריץ טסטים אחרי merge, ואם משהו נשבר — מחזיר אחורה אוטומטית

### ניהול עצמי
- Engineering Manager — מחליט מתי צריך להוסיף או להוריד סוכן
- Engineering Coach — מלמד את הצוות כל יום, משפר את הביצועים
- Product Manager — מקבל החלטות מוצר בלי לעצור את רומן

- לסטארטאפים שאין להם תקציב לצוות
- מחיר: $3,000–$8,000/חודש

- הם קונים יכולת להכפיל תפוקה
- טיקטים קטנים → הצוות של רומן

- לקוח משלם ₪20,000 לפרויקט
- רומן מוציא $500 ב-API
- שאר הכסף — רווח

### 4. הטמעה בתוך חברה
- החברה קונה את המערכת
- רומן מתקין + מלמד
- מחיר: $50,000–$100,000 חד פעמי
